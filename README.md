# Path of Exile 2 Toolkits (`poe2_tools`)

A collection of local, lightweight analytical tools and automation helpers built to optimize trading workflows, market flipping pathways, and mechanical calculations in Path of Exile 2.

# 🛠️ Sub-Tool Directory
## 📊 Exile Ledger
An offline graphical dashboard utilizing localized desktop persistence layers to evaluate arbitrage pathways.

Key Focus: Tracks profit margins while accounting for item-specific and raw currency trade market gold sink taxes.

Smart Matching: Solves multi-variable integer constraints to output exact bulk trading quantities accepted by the PoE2 trade site.
